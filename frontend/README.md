# Frontend

This folder contains the React-based frontend, built with **Vite** and **Bun**.

## Setup

```bash
cd frontend
bun install
bun dev
```

## File Structure

```md
frontend/
  components/     # React components
  js/             # Logic & utilities
  public/         # Static assets
  index.html      # Entry point
  main.jsx        # React root
  vite.config.js  # Vite config
```
